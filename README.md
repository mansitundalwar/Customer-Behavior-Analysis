 📊 Customer Behavior Analysis Dashboard

 🔹 Project Overview

This project analyzes customer purchasing behavior using Python, PostgreSQL, and Power BI. It covers the complete data analytics pipeline from data cleaning to visualization.

#🔹 Tech Stack

* Python (Pandas, NumPy)
* PostgreSQL (pgAdmin 4)
* Power BI
* Jupyter Notebook

🔹 Workflow

1. Data Cleaning using Python (Jupyter Notebook)
2. Data stored and queried in PostgreSQL
3. SQL queries used for analysis
4. Data visualization using Power BI dashboard

🔹 Key Insights

* Majority customers are non-subscribers (73%)
* Clothing category generates highest revenue
* Young adults contribute highest sales
* Average purchase amount is $59.76
